# Street Heat

Small GTA-style browser game prototype built with plain HTML, CSS, and JavaScript.

## Run

Open `index.html` in a browser.

## Controls

- `WASD` or arrow keys: move / drive
- `Shift`: sprint
- `Space`: shoot
- `E`: enter or exit a nearby car
- `R`: restart after getting busted

## Objective

Complete the three missions shown above the game canvas while surviving police pressure.
